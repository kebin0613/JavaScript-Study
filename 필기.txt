 1-3. 연산자
   -연산자 : 특정 연산을 하도록 하는 문자
   -산술 연산자 : +(덧셈), -(뺼셈), *(곱셈), /(나눗셈)
   -증감 연산자 :  a++(다음라인에서 연산) , --a(현재라인에서 연산)
   -대입 연산자 : 특정 값에 연산을 한 값을 바로 설정할 때 사용할 수 있는 연산자
                     a += 3; / a -= 3; / a *= 3; / a /= 3;
   -논리 연산자 : 불리언타입 (true, false)를 위한 연산자
      > ! (not) : 부정- 참이면 거짓, 거짓은 참
      > &&(and) : 둘다 참이거나 둘다 거짓일땐 참, 그외에는 거짓 
      > ||(or) : 둘중에 하나만 참이면 참, 그외에는 거짓
      >논리연산자의 우선순위 :  !(not) -> &&(and) -> ||(or)
   -비교연산자 : ==, ===, >, <, <=, >=
     > == : 두 값이 일치하는지 확인 
     > == : 두 값이 일치하는지 확인(타입도)
   -문자열 합치기 : "안녕" + "하세요" = "안녕하세요"

 1-4. 조건문 
   -if 문 : ~라면 ~를 해라
   -if else 문 : ~하다면 ~하고, 그렇지 않으면 ~를 해라
   -if else if 문 : 여러가지 조건에 따라 다른 작업을 해야할떄 사용
   -switch case문 : 특정 값이 무엇이냐에 따라 다른 작업을 하고 싶을 떄 사용
     > 각 case에서 코드 작성 마지막에 break; 를 꼭 작성해주어야 한다.
        안할시 다음case의 코드까지 실행해버린다
      >맨 마지막의 default 는 case에 없는 값일 경우 즉 else와 같은 그외 나머지경우를 의미
